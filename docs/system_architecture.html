03 // Unified XP system
One pool, two input channels, decay on inactivity
XP works like airline miles. Two ways to earn: ambassador activity (X posts, community work, content, introductions) and TG app activity (swaps, quests, streaks, referrals). Both feed the same score. Decay: if you stop contributing from either channel, your score decays. Active TG app users can boost their ambassador standing. Active ambassadors can boost their TG app standing. Phase 1: ambassador scoring only (live via Apify + daily admin review). Phase 2: TG app XP merges in as automated input. The bridge unifies when the app launches.
Ambassador input channel
X content quality, engagement, consistency. Community contribution. Telegram activity. Application quality. Admin override. Live quantitative scoring via Apify. Daily qualitative review by admin.
TG app input channel
Swap volume. Quest completions. Streak maintenance. Referrals. Partner quest engagement. News feed reads. Automated XP (Phase 2).
Score decay ensures XP reflects current commitment, not historical achievement. Miles expire if you don't fly.

Action-reward loops
Phase 1: Live scoring via Apify. Daily admin review. Phase 2: Automated quests with variable XP. Claim mechanics. Bonus drops. Variable reward cadence keeps engagement unpredictable.
Loss mechanics
Evangelist badge strips on consistency failure. Rank drops publicly. Score decays on inactivity. Leads lose title on cohort failure. The system handles enforcement automatically.
Identity formation
Public profiles. Track labels. Gold Evangelist badge. Bio integration. The ambassador identity becomes part of who they are.
Commitment deepening
Every maintained week = visible investment. Score history visible. Commitment deepens over time.
Consistency engine
7-day rolling comparison. Public trend (rising/falling/stable). Evangelist maintenance = trend must stay stable or rising. The leaderboard tells the story.
Social pressure
Public leaderboard. Visible Evangelist eligible list. Rising/falling indicators. Compete against people, not a bar.
Identity formation
Shared language. Insider access. Monthly rituals. Quarterly briefings. Evangelist as inner circle identity. Ambassadors carry the protocol identity as their own.
Admin control layer
Thresholds. Badge toggles. Overrides. Quest config. Eligibility rules. Decay rates. Everything feels organic. Admin configures the system.

Public trend indicators on leaderboard. Score decay on inactivity. Comms: badge is living recognition, not a promise. Loss is automatic, system-enforced, positively framed.
